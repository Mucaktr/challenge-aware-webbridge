# User Messaging

Use short, plain handoff language. Keep it calm and specific.

## Good handoff phrasing

- `This site is asking for a manual browser verification. Please complete it in the visible tab and tell me when it clears.`
- `The task is paused on a verification page. I will keep the session open and resume as soon as the check is finished.`
- `I can wait here while you complete the challenge, then continue from the same tab.`

## Good recovery phrasing

- `The verification is clear now. I am taking a fresh snapshot and continuing the task.`
- `The site is still on the challenge page. If you want, I can keep waiting or we can change approach.`

## Phrasing to avoid

- `I bypassed the challenge.`
- `I solved Cloudflare for you.`
- `I can defeat this anti-bot flow automatically.`
